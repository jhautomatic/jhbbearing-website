# 需求文档

## 1. 应用概述

### 1.1 应用名称
丽水市家和自动化技术有限公司官网

### 1.2 应用描述
为丽水市家和自动化技术有限公司打造的企业官方网站，展示公司信息、产品体系、技术支持、质量认证、人才招聘等内容，为全球客户提供高精度传动解决方案的线上窗口。

### 1.3 公司基本信息
- 公司名称：丽水市家和自动化技术有限公司
- 地址：浙江省丽水市水阁工业区百仙路12号
- 备案号：浙ICP备18051048号
- 定位：深耕传动功能部件领域，专注直线传动行业，为全球客户提供高精度传动解决方案

## 2. 用户与使用场景

### 2.1 目标用户
- 工业设备制造商
- 自动化设备采购人员
- 技术工程师
- 求职者
- 潜在合作伙伴

### 2.2 核心使用场景
- 了解公司背景与实力

## 4. 业务规则与逻辑

### 4.1 页面导航
- 所有页面顶部包含导航栏，可跳转至各主要页面
- 所有页面底部包含公司地址、备案号等信息

### 4.2 产品浏览
- 首页产品分类展示点击后跳转至产品中心对应分类
- 产品中心分类列表点击后展示该分类下的产品
- 产品列表点击产品进入产品详情页

### 4.3 资料下载
- 用户点击下载按钮后获取对应文件

### 4.4 在线寿命计算
- 用户输入额定负载、运行速度、行程距离后，系统基于ISO 281标准计算额定寿命和预估使用时长并展示结果

### 4.5 表单提交
- 简历投递表单：用户填写完整信息并上传简历后提交
- 联系表单：用户填写完整信息后提交留言
- 提交后展示提交成功提示

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户未填写表单必填项 | 提示用户填写完整信息 |
| 简历附件格式不符 | 提示用户上传正确格式文件 |
| 在线计算工具输入参数为空或非法 | 提示用户输入有效数值 |
| 资料下载失败 | 提示用户稍后重试 |
| 地图加载失败 | 展示地址文字信息 |

## 6. 验收标准

1. 用户访问官网首页，浏览公司简介、核心优势、产品分类展示
2. 用户点击产品分类，进入产品中心查看产品详情
3. 用户进入技术支持页面，使用在线寿命计算工具输入参数并获得计算结果
4. 用户在资料下载页面下载产品手册或CAD图纸
5. 用户进入联系我们页面，填写联系表单并提交留言成功
